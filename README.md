## Installing

Install 

```bash
npm ci
```


## Configuration

Set env vars
```sh
export CYPRESS_USERNAME=YOUR_USERNAME
export CYPRESS_PASSWORD=YOUR_PASSWORD
export CYPRESS_CONYUGE="Last Name First Name"
export CYPRESS_ADDRESS="Any address you want to use"
```

## Get Started

```sh
npm start
```
